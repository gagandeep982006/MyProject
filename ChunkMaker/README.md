🚀 Built a File Chunking, Compression & Encryption Tool in Java

I recently developed a Java-based utility that helps process large files efficiently by combining chunking, compression, encryption, and restoration into a single workflow.

🔹 Key Features:

• Split large files into multiple chunks

• Compress each chunk using GZIP to reduce storage requirements

• Encrypt compressed chunks using AES encryption for secure storage and transfer

• Process chunks concurrently using multithreading for better performance

• Restore files by automatically decrypting, decompressing, and merging chunks back into their original form


🛠️ Concepts Used:

• Core Java

• File Handling (Input/Output Streams)

• Multithreading

• GZIP Compression

• AES Encryption

• Custom Exceptions

• Interfaces and Strategy Pattern

• Object-Oriented Design Principles


This project helped me gain practical experience with file systems, data security, concurrency, and software architecture while building a real-world utility from scratch.

Looking forward to adding features such as metadata management, progress tracking, advanced compression algorithms, and a graphical user interface.
