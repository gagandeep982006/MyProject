# 📝 Magic Note – Simple Notes App

This is a simple Notes App built using HTML, CSS, and JavaScript.

The main purpose of this project is to create a small web application where users can write notes and keep them saved in the browser. You can add new notes, delete them whenever you want, and the data stays saved even after refreshing the page because it uses LocalStorage.

The interface is clean and responsive, and everything is handled using vanilla JavaScript without any external libraries.

## What this project does

- Create notes dynamically
- Delete notes
- Store notes in LocalStorage
- Load saved notes automatically on refresh
- Responsive layout using Flexbox

## Technologies Used

- HTML
- CSS
- JavaScript (DOM manipulation and event handling)

This project helped me practice working with the DOM, handling events, and managing browser storage.

Feel free to explore or improve it.
